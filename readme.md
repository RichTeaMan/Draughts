# Draughts AI Trainer
This project attempts to train an AI Draughts player.

The AI make decisions based on weights for board metrics.
These weights are honed with a generic algorithm.

## Cake Tasks
This project uses [Cake](https://cakebuild.net)!
* cake Clean
* cake Build
* cake Run-Unit-Tests
* cake Train
